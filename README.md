# ITHERA
Ithera — Ecosistema digital colaborativo para la planificación de viajes grupales en tiempo real. Plataforma web full-stack (Node.js + React.js + PostgreSQL + Socket.io + Redis) con sincronización  multiusuario, gestión de itinerarios, presupuestos compartidos e integración con APIs externas - Proyecto académico — IPN ESCOM · ADS · 5CM3 · Equipo 3.
