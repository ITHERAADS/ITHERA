# Extracted - Artefactos fuente

Esta carpeta queda reservada para artefactos fuente o exportaciones automaticas usadas como referencia.

## Importante

Los archivos extraidos automaticamente desde PDF pueden producir texto ilegible por problemas de encoding. Para evitar confusion, las extracciones `.txt` corruptas fueron retiradas de la documentacion principal.

La documentacion vigente debe consultarse en:

- [../README.md](../README.md)
- [../api/endpoints.md](../api/endpoints.md)
- [../architecture/README.md](../architecture/README.md)
- [../requerimientos/README.md](../requerimientos/README.md)
- [../casos-de-uso/README.md](../casos-de-uso/README.md)
- [../diagramas/README.md](../diagramas/README.md)

## Archivos permitidos aqui

- PDFs fuente.
- Exportaciones temporales que se van a revisar.
- Evidencias originales que aun no se han convertido en documentacion limpia.

## Reglas

- No enlazar desde el README principal a archivos crudos de esta carpeta.
- Si se extrae texto de un PDF, revisar encoding y legibilidad antes de subirlo.
- Si un archivo se convierte en documentacion oficial, moverlo al directorio correspondiente.
